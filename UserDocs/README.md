### Example Trackers

Ready-to-use tracker definitions you can copy to `.nimbalyst/trackers/`:

- [**character.yaml**](examples/character.yaml) - Track characters in books/movies/series
- [**recipe.yaml**](examples/recipe.yaml) - Organize your recipe collection
- [**research-paper.yaml**](examples/research-paper.yaml) - Track academic papers and research
- [**work-packet.yaml**](examples/work-packet.yaml) - Run spec-first Codex/Claude Code work with gates, skills, verification, and project memory

To use: Copy an example file to your workspace at `.nimbalyst/trackers/[filename].yaml` and restart Nimbalyst.
